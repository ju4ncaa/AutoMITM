## Requisitos

- **Sistema Operativo:** `Linux`
- **Herramientas necesarias:** `bettercap`, `arp-scan`

## Uso

### **Clonar el repositorio:**
```bash
git clone 
```
### Acceder a el repositorio descargada
```bash
cd AutoMITM
```
### Ejecutar herramienta
```bash
./mitm.sh
```
